# 🌤 Weather App Pro

Une application météo moderne développée avec **React + TypeScript + Vite**.

---

## 🚀 Fonctionnalités

- Recherche météo par ville
- Géolocalisation automatique
- Température en temps réel
- Humidité et vent
- Mode sombre / clair
- Conversion °C / °F
- Autocomplétion des villes

---

## 🛠 Technologies

- React
- TypeScript
- Vite
- WeatherAPI
- Recharts
- LocalStorage
